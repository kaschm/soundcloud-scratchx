SoundCloud API in ScratchX

About

Using the SoundCloud extension, you can play music from SoundCloud directly inside of ScratchX using a few simple blocks. Make your own playlists, music videos, or animated stories with music from SoundCloud!

ScratchX is a playground for experimental extensions using the familiar blocks and interface from Scratch, an online, block-based programming tool and community. You can learn more about ScratchX by visiting scratchx.org, and more about the SoundCloud API via their Developer Documentation page.

Scratch is a project of the Lifelong Kindergarten at the MIT Media Lab.

How to Use

There are six new blocks with this extension that enable you to find and play songs from SoundCloud using the SoundCloud API.

Load Song ID

Use this block to load the song from SoundCloud, using its SongCloud ID. You can use the "Get ID" block, below, to find the ID from the Song URL. Songs that do not exist will throw an error.

Get Song ID

Use this block to find the song ID from SoundCloud. Paste the URL of the song into the blank area.

Play Song

This block will play the song whose ID has been loaded via the Load Song ID block.

Pause Song

This block will pause the song that is currently playing. Clicking the Play button will resume play at the point it was paused.

Stop Song

Using this block will stop the current song from playing. Playing the song again will cause it to start from the beginning.

Get Title of Song

This block will return the Title of the Song based on the ID pasted in the blank area.
